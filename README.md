# WIMS (Warehouse Inventory Management System)

Simple WIMS built with Laravel, Inertia.js, React and Material-UI.

## Features

- Transactional Data:
  - Purchase Order
  - Sales Order
  - Goods Receipt
  - Order Delivery
- Full Data:
  - Product
  - Warehouse
  - Location
  - Client
  - Vendor
  - User
- Dashboard (UI only)
  - Throughput
  - Stock Aging
  - Outstanding Delivery
  - Lead Time
